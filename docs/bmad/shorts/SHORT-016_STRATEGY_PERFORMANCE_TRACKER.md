# SHORT-016: Strategy Performance Tracker

**Parent**: FX-002 (Kelly Position Sizing)
**Estimated Effort**: 4 hours
**Priority**: HIGH

## Objective

Track strategy performance metrics (win rate, avg profit/loss, consecutive wins/losses) needed for Kelly fraction calculation.

## Acceptance Criteria

- [ ] AC-1: Track individual trade outcomes (win/loss/breakeven)
- [ ] AC-2: Calculate win rate over configurable period
- [ ] AC-3: Calculate average profit per winning trade
- [ ] AC-4: Calculate average loss per losing trade
- [ ] AC-5: Track consecutive wins and losses
- [ ] AC-6: Persist performance history
- [ ] AC-7: Support rolling window calculations
